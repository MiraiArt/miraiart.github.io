JS
                
                    
                    //Init alert
                    $.MA_Alert({
                        message: 'Please defined a value !',
                        title: '<i class="fas fa-exclamation-triangle"></i> Error !',
                        classHead: 'bg-purple',
                        backgroundClose: false,
                        closeBtn: true,
                        buttons: {
                            btnClose: {
                                class: 'bg-blue',
                                name:'dqzdq',
                                action: function () {
                                }
                            },
                            btnRemove: {
                                class: 'bg-red',
                                action: function () {
                                }
                            }
                        }
                    });

Options

Name Type Default value Description
transitionDuration Number 500 It's transition duration for the alert when it's open or close, in milliseconds.
title String null It's alert title.
message String null It's message display in alert.
icon String null It's icon display on head of alert.
classContainer String null Class apply on .alert-container.
classBackground String null Class apply on .alert-background.
classAlert String null Class apply on .alert.
classHead String null Class apply on .alert-head.
classBody String null Class apply on .alert-body.
classFoot String null Class apply on .alert-foot.
backgroundClose Boolean false If it's set to true alert can be close with background.
closeIcon String null It's icon for close alert.
closeBtn Boolean false If it's true the button will be displayed.
buttons Json {} It's possible to add buttons on foot with href, class, name, and action properties.
