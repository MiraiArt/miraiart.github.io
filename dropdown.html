Dropdown

Dropdown can be used for show and hide a selectable list of link for example

If you want to change button background color of dropdown you can use bg-* class.

And for change size of button use the class btn-small or btn-large.

Dropdown default
  • Item 1
  • Item 2
  • Item 3
  • Item 4
Dropdown large
  • Item 1
  • Item 2
  • Item 3
  • Item 4
Dropdown normal
  • Item 1
  • Item 2
  • Item 3
  • Item 4
Dropdown small
  • Item 1
  • Item 2
  • Item 3
  • Item 4
Dropdown disabled
  • Item 1
  • Item 2
  • Item 3
  • Item 4
HTML
                

                    <!-- Default Dropdown large -->
                    <div class="dropdown">
                        <a class="btn btn-dropdown btn-large" href="javascript:void(0);">Dropdown default</a>
                        <ul class="dropdown-list">
                            <li class="dropdown-item"><a class="dropdown-link" href="#">Item 1</a></li>
                            <li class="dropdown-item"><a class="dropdown-link" href="#">Item 2</a></li>

                            <li class="dropdown-item"><a class="dropdown-link" href="#">Item 4</a></li>
