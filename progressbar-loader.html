Progress bar / Loader

Progress bar

Show head progress bar
  • HTML
  • JS
                
                    <!-- Progress bar -->
                    <div class="progress-bar">
                        <div class="progress bg-blue" style="width: 15%;"></div>
                    </div>
                    
                    <!-- Head Progress bar -->
                    <div class="head-progress">
                        <div class="progress-bar">
                            <div class="progress bg-blue" style="width: 100%;"></div>
                        </div>
                    </div>
                
            
                
                    //Head progress bar animation
                    $('.head-progress .progress-bar').MA_Progress({
                        transitionDuration: 10000,
                        startAt: '0%',
                        endAt: '100%',
                        hideAtEnd: true
                    });

Options

Name Type Default value Description
transitionDuration Number 500 It's duration for progress bar, in milliseconds.
startAt Number 0% Start width of progress bar.
endAt Number 100% End width of progress bar.
hideAtEnd Boolean true Hide progress bar when it's finished to load.

Loader

Show page loader
  • HTML
  • JS
                
                    <!-- Loader -->
                    <div class="loader border-green"></div>
                    
                    <!-- Page loader -->
                    <div class="page-loader">
                        <div class="loader border-red"></div>
                    </div>
                
            
                
                    //Show / hide page loader
                    $('.page-loader').fadeIn();
                    $('.page-loader').fadeOut();
