Toast

Toast it's a small message, you can display after or before an action.

Colored toast

Show toast
JS
                
                    
                    $.MA_Toast({
                        message: 'Do you want to go at home page ?',
                        icon: '<i class="fas fa-exclamation-triangle"></i>',
                        class: 'bg-blue',
                        closeable: true,
                        toastClose: {
                            name: '<i class="fas fa-times"></i>',
                        },
                        toastBtn: {
                            name: 'Go to homepage',
                            href: 'index.html',
                            action: function () {
                            }
                        }
                    });

Options

Json name Type Default value Description
message String null It's toast message, you can put HTML inside.
class String null It's class append to the toast.
icon String null This is icon of title for toast.
duration Number 3000 It's duration of toast before fade out.
toastClose.class
toastBtn.class		 String null It's class for button of toast.
toastClose.name
toastBtn.name		 String null It's a name for button, you can put HTML if you wan't.
toastBtn.href String null It's possible to add a link to button.
toastBtn.action Function null You can add action on button.
