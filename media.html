Media

Img zoom

  • HTML
  • JS
                

                    <!-- Img -->
                    <img class="img-zoom-one" src="https://via.placeholder.com/1920x1080" data-description="MiraiArt image description">

                
            
                
                    //Load ImgZoom
                    $('.img-zoom-one').MA_imgZoom({
                        button: {
                            close: {
                                name: '<i class="fas fa-times"></i>',
                            }
                        }
                    });

Img zoom group

  • HTML
  • JS
                

                    <!-- Img -->
                    <img class="img-zoom-group" data-group="imgZoom" src="https://via.placeholder.com/1700x980" data-description="MiraiArt image description image 1">
                    <img class="img-zoom-group" data-group="imgZoom" src="https://via.placeholder.com/320x280" data-description="MiraiArt image description image 2">

                
            
                
                    //Load ImgZoom group
                    $('.img-zoom-group').MA_imgZoom({
                        button: {
                        arrowLeft: {
                            name: '<i class="fas fa-arrow-left"></i>',
                            class: 'bg-blue',
                        },
                        arrowRight: {
                            name: '<i class="fas  fa-arrow-right"></i>',
                            class: 'bg-blue',
                        },
                        close: {
                            name: '<i class="fas fa-times"></i>',
                            class: 'bg-blue',
                        }
                    }
                    });

Options

imgItemSlideClass String null Add class on .img-zoom-slide
imgItemWrapperClass String null Add class on .img-zoom-img-wrapper
imgItemDescriptionClass String null Add class on .img-zoom-description
button.arrowLeft.class
button.arrowRight.class
