Button

The button can be smaller or larger and it's possible to add bg-* class for change background color.

Color

Button default Button orange Button red Button pink Button purple Button blue Button green Button black

Size

Button small Button normal Button large

Disabled / Active

Button disabled Button active

Pagination

1 2 3 4
HTML
                

                    <!-- Button color -->
                    <a class="btn" href="#">Button default</a>
                    <a class="btn bg-orange" href="#">Button orange</a>
                    <a class="btn bg-red" href="#">Button red</a>
                    <a class="btn bg-pink" href="#">Button pink</a>
                    <a class="btn bg-purple" href="#">Button purple</a>
                    <a class="btn bg-blue" href="#">Button blue</a>
                    <a class="btn bg-green" href="#">Button green</a>
                    <a class="btn bg-black" href="#">Button black</a>

                    <!-- Button size -->
                    <a class="btn bg-pink btn-small" href="#">Button small </a>
                    <a class="btn bg-black" href="#">Button normal </a>
                    <a class="btn bg-green btn-large" href="#">Button large </a>
                    
                    <!-- Button disabled / active -->
                    <a class="btn bg-black disabled" href="#">Button disabled</a>
                    <a class="btn bg-purple active" href="#">Button active</a>

                    <!-- Pagination -->
                    <div class="flex-group">
                        <a class="btn bg-blue"><i class="fas fa-chevron-left"></i></a>
                        <a class="btn bg-blue no-shadow">1</a>
                        <a class="btn bg-blue active">2</a>
                        <a class="btn bg-blue no-shadow">3</a>
                        <a class="btn bg-blue no-shadow disabled">4</a>
                        <a class="btn bg-blue"><i class="fas fa-chevron-right"></i></a>
                    </div>
