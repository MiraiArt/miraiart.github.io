                    <!-- Auto-close accordion -->
                    <div class="accordion accordion-close">
                        <div class="accordion-item">
                            <div class="accordion-head bg-red">Accordion 1</div>
                            <div class="accordion-body">Accordion body 1</div>
                        </div>
                        <div class="accordion-item">
                            <div class="accordion-head bg-red">Accordion 2</div>
                            <div class="accordion-body">Accordion body 2</div>
                        </div>
                        <div class="accordion-item">
                            <div class="accordion-head bg-red">Accordion 3</div>
                            <div class="accordion-body">Accordion body 3</div>
                        </div>
                    </div>

                
            
                
                    //Init auto-close accordion
                    $('.accordion-close').MA_Accordion({ closeOtherItem: true });

Options

Name Type Default value Description
transitionDuration Number 200 I'ts duration for display and hide accordion, in milliseconds.
closeOtherItem Boolean false If it's set to true they will close all other item in accordion, on click.
