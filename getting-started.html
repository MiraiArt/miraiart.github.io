Getting started

Download

Mirai Art can be installed in your project in different ways.

Jquery is required for Miraiart for working properly and moment.js is just required for Datetimepicker.

Github   NPM   CDN Direct Download  

HTML example

Setting up the html page with MiraiAirt by using CDN or embedding files into your project

  • Html
  • With CDN
                
                    <!DOCTYPE html>
                    <html>
                    
                        <head>
                            <meta charset="UTF-8">
                            <title>MiraiArt</title>
                            <meta content="width=device-width, initial-scale=1" name="viewport">

                            <!--Import mirai-art.css -->
                            <link rel="stylesheet" type="text/css" href="css/mirai-art.min.css">
                        </head>
                    
                        <body>

                            <!-- Jquery is required for working -->
                            <script type="text/javascript" src="js/jquery-3.4.1.min.js"></script>

                            <!-- Moment.js is required for Datetimepicker -->
                            <script type="text/javascript" src="js/moment.min.js"></script>
                            
                            <!--Import mirai-art.js -->
                            <script type="text/javascript" src="js/mirai-art.min.js"></script>
                        </body>
                    
                    </html>
