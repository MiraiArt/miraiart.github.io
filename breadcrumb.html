Breadcrumb

Breadcrumb it's an element where you can put parent path of the webpage

It's possible to use background color bg-* class on breadcrumb to change is color.

Default breadcrumb

  • Home
  • Breadcrumb
HTML
                

                    <!-- Default Breadcrumb -->
                    <ul class="breadcrumb-list">
                        <li class="breadcrumb-item"><a href="/">Home</a></li>
                        <li class="breadcrumb-item">Breadcrumb</li>
                    </ul>

Orange breadcrumb

  • Menu 1
  • Menu 2
  • Menu 3
  • Menu 4

                    <!-- Breadcrumb with bg-orange -->
                    <ul class="breadcrumb-list bg-orange">
                        <li class="breadcrumb-item"><a href="#">Menu 1</a></li>
